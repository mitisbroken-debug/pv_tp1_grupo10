objetivos del proyectos

1.Optimización de Consultas Académicas: El objetivo de este proyecto es mejorar el rendimiento de la base de datos académica, reduciendo los tiempos de respuesta en la búsqueda de información de alumnos y materias. Se busca lograr un acceso más rápido y eficiente a los datos. Para ello, se aplican técnicas como indexación y optimización de consultas SQL, permitiendo una mejor organización y un sistema más ágil y confiable.

2. Automatización de Planillas de Asistencia: Este proyecto tiene como objetivo automatizar el registro de asistencias mediante macros en Excel, evitando la carga manual y reduciendo errores. Además, permite calcular automáticamente los porcentajes de asistencia y generar reportes, facilitando el control y seguimiento de los alumnos.

3. Script de Respaldo de Servidores: El objetivo es crear un script en Bash que realice copias de seguridad automáticas de archivos importantes en servidores Linux, asegurando la protección de los datos. También se busca que el proceso sea configurable y eficiente, permitiendo programar respaldos periódicos y mejorar la gestión del sistema.

4. Análisis de Rendimiento Estudiantil: Este proyecto busca analizar el rendimiento académico de los estudiantes mediante un programa en Python que genera gráficos y estadísticas a partir de sus notas. De esta forma, se facilita la interpretación de los datos y se apoya la toma de decisiones para mejorar el desempeño académico.

5. Sistema de Gestión Bibliotecaria: El objetivo es diseñar una base de datos para gestionar préstamos y devoluciones de libros de manera organizada y eficiente. Además, permite llevar un mejor control de usuarios y disponibilidad de ejemplares, optimizando el funcionamiento de la biblioteca.
